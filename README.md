# Keyloop — Intelligent Inventory Dashboard

> Scenario B (Supply) of the Keyloop technical assessment.
> Frontend implementation with mocked backend.

## Status

Project scaffolded. No business logic implemented yet.

## Stack

- Vite + React 19 + TypeScript (strict)
- TanStack Query — server state, refetch on focus, polling
- MSW — mock REST backend
- Tailwind CSS — styling
- React Hook Form + Zod — form validation
- Vitest + React Testing Library — testing
- date-fns, lucide-react, sonner

## Scripts

```bash
npm run dev        # start dev server
npm run build      # type-check + production build
npm run test       # run tests in watch mode
npm run test:run   # single test run
npm run lint       # eslint
```
